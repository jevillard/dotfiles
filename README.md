# Jérémy's dotfiles

These are my dotfiles. Take anything you want, but at your own risk.

## Installation

```bash
git clone git://github.com/jevillard/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
chmod +x bootstrap.sh
./bootstrap.sh
```
